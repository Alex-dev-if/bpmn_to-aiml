# Ari: Um chatbot que conversa sobre depressão
Autor: Giseldo da Silva Neo

Este chatbot utiliza 
* aiml
* flask
* spacy 

# Configurações
* pip install -r requirements.txt
* pip install aiml
* pip install flask
* pip install spacy
* python -m spacy download en_core_web_sm
* python -m spacy download pt_core_news_sm
